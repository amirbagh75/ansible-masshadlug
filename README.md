Ansible چیست؟

Ansible را اصطلاحا software provisioning, configuration management و application-deployment tool مینامند. خودمونی ترش این میشه که با Ansible میشه خیلی راحت تر تعداد زیاد سرور یا حتی ۱ عدد سرور رو مدیریت کرد. مثلا یکبار برای همیشه نصب اصولی nginx رو یاد بگیریم و با ansible پیاده کنیم و همیشه و همه جا ازش استفاده کنیم. یا مثلا میتونیم باهاش روی ۱۰ تا سرور شرکت چک کنیم اگر نسخه کرنل زیر فلان بود به مدیرسیستم ها خبر بده که فلان کار را بکنند یا حتی خودش یک کاری رو بکنه.
